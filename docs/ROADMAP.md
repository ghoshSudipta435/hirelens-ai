# Product and Engineering Roadmap V2

## Phase 0: Foundation

- Finalize V2 docs
- Initialize monorepo structure
- Set up TypeScript, linting, formatting, and testing
- Configure Prisma and PostgreSQL
- Set up environment variable strategy

## Phase 1: Core MVP Backend

- Auth module with JWT
- User profile endpoints
- Resume upload and parsing pipeline
- Job posting CRUD
- Application workflow
- Deterministic match scoring service

## Phase 2: Core MVP Frontend

- Auth screens
- Student dashboard
- Recruiter dashboard
- Resume upload flow
- Job posting creation flow
- Match result and ranking views

## Phase 3: AI-Assisted Features

- Interview question generation
- Explainability improvements for matching
- Provider abstraction for OpenAI and open-source model options

## Phase 4: Hardening

- Integration tests
- Rate limiting
- Better observability
- Async job support if parsing latency requires it
- Archive and retention behavior

## Deferred Beyond MVP

- Refresh tokens
- Password reset
- Admin panel
- Team recruiter accounts
- Advanced analytics
- Semantic embeddings and vector search
- Browser extension or ATS integrations
